# Security Policy

## Supported Versions

This repository is actively maintained on the default branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by email to:

- janmedina14@gmail.com

Please include:

- A clear description of the issue.
- Steps to reproduce.
- Impact assessment.
- Any suggested remediation.

We will acknowledge receipt and evaluate the report as quickly as possible.

## Security Maintenance

To reduce security risk over time:

- Dependencies are monitored with Dependabot (`bundler`, `npm`, and `github-actions`).
- Public secrets should never be committed to this repository.
- Automated dependency updates should be reviewed and merged regularly.
